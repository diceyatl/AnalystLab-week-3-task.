# AnalystLab Africa | Week 3 SQL Analysis

## Project Overview

This project contains my Week 3 SQL assignment from the AnalystLab Africa Data Analytics Internship (Batch B).

Using SQL Server Management Studio (SSMS), I analyzed two datasets:

- Chinook Database (music store database)

The objective was to answer business questions, explore customer behavior, analyze revenue trends, and strengthen practical SQL skills through real-world querying.

## Business Questions Answered

### Chinook Database

- Identified the highest revenue-generating countries.
- Ranked customers by lifetime value.
- Identified the best-selling artists and genres.
- Measured average invoice value by country.
- Found tracks that have never been purchased.
- Analyzed customer purchase behavior.
- Evaluated support representative performance.

### Sample Sales Dataset

- Analyzed yearly and quarterly revenue trends.
- Identified top customers and countries.
- Compared revenue across product lines.
- Evaluated deal size performance.
- Calculated year-to-date revenue.
- Measured month-over-month revenue changes.
- Ranked products within product categories.

---

## Key Insights

- USA, Canada, and France generated the highest revenue in the Chinook database.
- Chile recorded the highest average invoice value despite not leading total revenue.
- Over 1,500 tracks had never been purchased, highlighting opportunities for better product promotion.
- Iron Maiden generated the highest artist revenue.
- Classic Cars was the strongest-performing product line in the sales dataset.
- Q4 consistently generated the highest revenue across all years.
- Medium-sized deals produced the highest overall revenue across every country.
